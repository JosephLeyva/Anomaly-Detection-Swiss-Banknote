# Anomaly-Detection-Swiss-Banknote
Implementación de algunos métodos para la Detección de Outliers en un conjunto de datos para la detección de billetes falsos

Este trabajo utilizó el conjunto de datos de Kaggle sobre [Detección de Billetes Falsos](https://www.kaggle.com/chrizzles/swiss-banknote-conterfeit-detection)
El conjunto de datos tiene 6 columnas donde indican las medidas de los billetes y una columna que indica si es falso (1) o real (0). Se tienen 100 billetes verdaderos y 100 billetes falsos.
